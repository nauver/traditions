# CoR Traditions Challenge — V3 avec 400 traditions intégrées

## Contenu

- Visuel repris de la version 1.3.
- Base de 400 traditions intégrée dans `questions-400.js`.
- Tirage aléatoire de 30 questions à chaque partie.
- Maximum 2 questions par pays.
- 4 réponses par question.
- Mode classique et mode duel équipes.
- Timer de 20 secondes.
- Aucun backend, aucun build.

## Fichiers à uploader à la racine du repo GitHub

- `index.html`
- `style.css`
- `app.js`
- `questions-400.js`
- `assets/CoR_logo.png`
- `README.md`
- `metadata.json`

## Déploiement

GitHub > Settings > Pages > Deploy from branch > main > /root.
